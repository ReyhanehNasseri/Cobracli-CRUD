

# CRUD Application in Go with Cobra CLI

This project is a simple CRUD (Create, Read, Update, Delete) application developed using the Go programming language and the Cobra CLI library. The purpose of this project is to demonstrate the implementation of basic CRUD operations using a command-line interface.

## Features

- Create, Read, Update, and Delete records
- User-friendly command-line interface using Cobra
- Input validation and error handling

## Technologies Used

- Programming Language: Go
- CLI Framework: Cobra
